# Spring Databases Embedded - Sample projects

### Run
#### hsqldb:
```
cd hsqldb
mvn clean verify
```

#### postgres:
```
cd postgres
mvn clean verify
```

#### sqlite:
```
cd postgres
mvn clean verify
```
